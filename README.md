# Morphological-Analysis-on-Indian-Languages

For Part of Speech Tagger (POS Tagger), we have created lists of Nouns, Verbs, Adjectives and Adverbs. 
Example: ਮੰਡਾ (Boy) in Punjabi is a Noun and we will select this word and would search in our lists. If the word is found in our Noun List we will classify it as Noun
